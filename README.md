## 🧪 Como Rodar o Projeto

```bash
# Clone o repositório
git clone https://github.com/GuiiiChaves/livraria-digital.git

# Acesse o diretório
cd livraria-digital

# Instale as dependências
npm install

# Inicie o frontend (Next.js)
npm run dev
```
